# 仓库管理系统简介 python921

## 项目概述

仓库管理系统是一款基于Python开发的Web应用程序，旨在帮助企业和组织高效管理库房、商品、出库和入库信息。系统采用B/S架构，前端使用Vue.js，后端采用Django框架，数据库使用MySQL。

## 技术栈

- **后端**: Python, Django
- **前端**: Vue.js
- **数据库**: MySQL
- **架构**: B/S
